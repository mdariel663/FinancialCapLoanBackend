Documentación de la API - Sistema de Préstamos

API para la gestión en tiempo real de un sistema de préstamos.

Autenticación

POST /auth/login

Resumen: Autenticación de usuarios

Request Body:
{
    "username": "string",
    "password": "string"
}

Respuestas:

  • 200: Inicio de sesión exitoso
  • 401: Credenciales incorrectas

POST /auth/register

Resumen: Registro de nuevos usuarios

Request Body:
{
    "username": "string",
    "password": "string",
    "role": "string"
}

Respuestas:

  • 201: Registro exitoso
  • 400: Datos inválidos

Usuarios

GET /users

Resumen: Obtener lista de usuarios

Respuestas:

  • 200: Lista de usuarios recuperada

GET /users/{userId}

Resumen: Obtener detalles de un usuario específico

Parameters:
userId: string

Respuestas:

  • 200: Usuario encontrado
  • 404: Usuario no encontrado

PUT /users/{userId}

Resumen: Editar información de un usuario

Parameters:
userId: string

Request Body:
{
    "username": "string",
    "role": "string",
    "permissions": ["string"]
}

Respuestas:

  • 200: Usuario actualizado
  • 404: Usuario no encontrado

DELETE /users/{userId}

Resumen: Eliminar un usuario

Parameters:
userId: string

Respuestas:

  • 204: Usuario eliminado
  • 404: Usuario no encontrado
